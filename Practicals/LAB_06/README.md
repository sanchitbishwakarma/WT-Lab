# Student Registration Form

Create an HTML student registration form with the following fields:

## Personal Information
- **First Name** - Text input field
- **Last Name** - Text input field

## Contact Details
- **Gender** - Radio buttons
- **Phone** - Number input field
- **Email** - Email input field

## Family Information
- **Father's Name** - Text input field
- **Mother's Name** - Text input field

## Academic Information
- **Program** - Dropdown selection
    - BCA
    - BIT
    - BCE
    - BBA

## Additional Information
- **Interests** - Checkboxes
    - Cybersecurity
    - Data Science
    - AI
- **About Yourself** - Multiline text area
- **Hobby** - Multiple selection dropdown
    - Singing
    - Dancing
    - Football
    - Chess

## Form Actions
- **Submit** - Button
- **Reset** - Button
